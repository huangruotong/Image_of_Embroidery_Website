Modes Explained

Line Mode

Based on Skeletonization algorithm, which thins images to single-pixel-wide line skeletons.

Technical Principle:

Progressively "erodes" the image through morphological operations, preserving the centerline structure

Use Cases:

Handwritten signatures, sketches, simple line art

Adjustable Parameter:

Line Precision - Controls line thickness and detail preservation (0-100)

Canny Edge Mode

Uses the classic Canny Edge Detection algorithm to precisely identify image edges through gradient calculation.

Technical Principle:

Gaussian blur → Sobel operator gradient calculation → Double threshold detection → Edge linking

Use Cases:

Landscape photos, portraits, complex object contours

Adjustable Parameters:

Threshold Low (0-255): Pixels below this value are suppressed
Threshold High (0-255): Pixels above this value are confirmed as edges

Raster Fill Mode

Uses Raster Fill Strategy, scanning tonal regions and converting them into adaptive stitch density.

Technical Principle:

Row-wise serpentine scan → brightness-based spacing → adaptive stitch density in darker areas

Use Cases:

Portrait photos, grayscale shading, and broad filled areas

Adjustable Parameters:

Row Spacing (1-16): Controls scan line density
White Threshold (120-250): Skips very bright pixels

Frequently Asked Questions

  • • Simple line drawings (signatures, sketches) → Choose Line mode
  • • Complex photos (landscapes, portraits) → Choose Canny mode
  • • Portrait photos (faces, tonal shading) → Choose Raster mode

Possible reasons and solutions:

  • Low image quality: Use high-resolution, clear images
  • Incorrect parameters: Try adjusting sliders and observe the real-time preview
  • Wrong algorithm: Switch to another mode
  • Complex background: Consider removing the background with an image editor first
